# Programa de Locacao com Opcao de Compra - Uny FleetCom<sup>®</sup>

Apresentacao em HTML com navegacao por slides, layout responsivo e recursos visuais para exibicao em desktop, tablet e celular.

## Conteudo principal

- Arquivo principal: `apresentacao_unificada.html`
- Imagens e midias: `assets/ppt_media/`
- Arquivos de apoio (slides separados): `capa.html`, `funcionamento_contrato.html`, etc.

## Como abrir

1. Abra o arquivo `apresentacao_unificada.html` no navegador.
2. Use os botoes de navegacao para:
   - ir para o inicio
   - voltar um slide
   - avancar um slide
   - ir para o fim

## Recursos implementados

- Transicao entre slides com efeito fade
- Contador com numero anterior, atual e proximo
- Layout responsivo para diferentes tamanhos de tela
- Ajustes de leitura e posicionamento para mobile
- Nota legal de marca registrada no ultimo slide
